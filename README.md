# Rhova

Rhova is a real-time quiz application similar to Kahoot. It is open source, requires no user accounts, and does not store any user data.

You can create a quiz, launch it, and allow participants to join by scanning a QR code or entering a room code with their phones. The application is designed to be simple and intuitive to use.

![A quick demo of Rhova](docs/demo.gif)

## Official Deployment

If you do not want to deploy your own instance of Rhova or simply want to try it out, you can use the official deployment: https://rhova.app

## Contact

For questions, suggestions, bug reports, feature requests, or assistance with deploying Rhova on your own servers, contact: [contact@rhova.app](mailto:contact@rhova.app)

## Run Locally

Clone the repository (or fork it if you plan to modify it):

```bash
git clone git@github.com:Cereal38/rhova.git
```

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.
